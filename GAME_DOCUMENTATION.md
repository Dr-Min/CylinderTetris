## 1. 게임 개요 (Game Overview)
플레이어는 사이버 공간의 **해커**가 되어 여러 **스테이지(섹터)**를 점령해 나가며, 최종 목표인 **보스 코어(Core Nexus)**를 정복하는 것이 목표입니다. 각 스테이지에서 적 바이러스들의 공격을 막아내고, 최대 페이지를 생존한 뒤 **테트리스 방어전**을 통해 스테이지를 점령합니다.

---

## 2. 게임 모드 및 핵심 메커니즘

### 🛡️ A. 디펜스 모드 (Defense Mode - Main)
게임의 기본 대기 화면이자 메인 방어 모드입니다.

*   **시점**: Top-down 2D View (Canvas)
*   **목표**: 중앙의 **CORE** 체력(HP)이 0이 되지 않도록 사수.
*   **주요 요소**:
    1.  **CORE (기지)**:
        *   화면 중앙에 위치. HP 100으로 시작.
        *   파괴 시 **SYSTEM CRITICAL FAILURE**와 함께 게임 오버.
    2.  **TURRET (포탑)**:
        *   CORE 주변을 회전하며 적을 자동 조준 및 사격.
        *   적이 없으면 360도 시계방향으로 자동 회전.
        *   **수동 발사**: 클릭/터치/스페이스바로 추가 발사 가능.
        *   탄환은 **랜덤 ASCII 문자**로 표시 (초록색).
    3.  **ENEMY (바이러스)**:
        *   화면 밖에서 생성되어 CORE를 향해 돌진.
        *   충돌 시 CORE에 데미지를 입힘.
        *   피격/사망 시 **글리치 파티클** 효과 (RGB 분리, 플리커).
    4.  **ALLIED VIRUS (아군 바이러스)**:
        *   파란색 원형, 적에게 몸통박치기 공격.
        *   사망 시 2초 후 자동 리스폰.
        *   점령된 스테이지: 10마리 유지.
    5.  **SHIELD SYSTEM (핵심 전략)**:
        *   **ACTIVE**: 파란색 역장. 적 튕겨냄. CORE 무적.
        *   **OFF**: 역장 해제. 자원(Data) 획득 가능. 피격 위험.
        *   **CHARGING**: 켜는 중 (2초). 점선 가속 회전.
        *   **DISCHARGING**: 끄는 중 (1초). 점선 전환.
        *   **BROKEN**: 내구도 0. 5초 수리 후 재충전.
        *   **색상 그라데이션**: HP 100%=파란색, 1%=빨간색.

### 🧱 B. 마이닝/점령 모드 (Tetris Mode)
스테이지 점령 시 발동하는 테트리스 방어전입니다.

*   **시점**: 3D Cylinder View (Three.js) - 원통형 테트리스
*   **진입 조건**: 최대 페이지 도달 후 `>>> CONQUER THIS SECTOR <<<` 선택
*   **목표**: 3줄 클리어 + 강화 페이지 3개 생존
*   **특징**:
    *   점령 중 실드 사용 불가 (파괴 애니메이션).
    *   미니 디펜스 패널로 방어 상황 확인.
    *   테트리스 클리어 후 강화 페이지 생존 필요.

## 3. 스테이지/맵 시스템

### 📍 맵 구조

      ┌─────┴─────┐        │        ┌─────┴─────┐
      │  Sector   │        │        │  Sector   │  ← 상단 스테이지

            │              │              │
            └──────────────┼──────────────┘
                           │
                    ┌──────┴──────┐
                    │  [FARMING]  │  ← 파밍존 (점령 불가)
                    │    Zone     │
                    └──────┬──────┘
                           │
            ┌──────────────┼──────────────┐
            │              │              │
      ┌─────┴─────┐        │        ┌─────┴─────┐
      │  Sector   │        │        │  Sector   │  ← 하단 스테이지
      │   Alpha   │        │        │   Beta    │

            │              │              │
            └──────────────┼──────────────┘
                           │
                    ┌──────┴──────┐
                    │ [SAFE ZONE] │  ← 안전영역 (시작점)

### 스테이지 정보

| ID | 이름 | 타입 | 페이지 | 난이도 스케일 | 적 유형 | 설명 |
|----|------|------|--------|--------------|---------|------|
| 0 | Safe Zone | 안전 | 없음 | - | basic | 시작점, Home Base |
| 1 | Sector Alpha | 점령 | 5 | 1.5x | basic, fast | 하단 좌측, 진입점 |
| 2 | Sector Beta | 점령 | 5 | 1.5x | basic, tank | 하단 우측, 진입점 |
| 3 | Data Mine | 파밍 | 없음 | - | basic, fast, tank | 점령 불가, 경유지 역할 |
| 4 | Sector Gamma | 점령 | 12 | 1.0x | fast, tank, elite | 상단 좌측, 고위협 |
| 5 | Sector Delta | 점령 | 12 | 1.0x | fast, tank, elite | 상단 우측, 고위협 |
| 6 | Core Nexus | 보스 | 15 | 0.8x | elite, boss | 최종 보스 (느린 난이도 증가) |

### 점령 시스템
- 최대 페이지 생존 → `∞ READY` 표시
- `>>> CONQUER THIS SECTOR <<<` 선택
- 테트리스 3줄 클리어 + 강화 페이지 3개 생존
- 성공 시: 🚩 깃발 + ⭐ 별 모양 방어막 + 아군 10마리

### 스테이지 접근 규칙
1. **안전 영역(Safe Zone)**: 항상 접근 가능
2. **직접 연결 스테이지**: Safe Zone에서 연결된 Alpha/Beta는 항상 접근 가능
3. **점령 스테이지**: 점령한 스테이지에서 연결된 스테이지 해금
4. **파밍 스테이지**: 점령 불가, 방문 시 연결 스테이지 해금 (경유지 역할)
5. **미점령 스테이지**: 통과 이동 불가

### 스테이지 병합 (Conquest Manager)
점령한 스테이지는 자동 병합되어 아군을 강화합니다:
- **2개 스테이지 병합** → 1개 강화 주둔지로 변환
- **병합당 아군 레벨 +1** (체력/능력 증가)
- **채굴량 공식**: `(10 + (레벨-1)*5) × (점령지 + 병합스택*1.5)`

| 상태 | 아군 수 | 체력 |
|------|---------|------|
| 기본 (미점령) | 3마리 | 10 |
| 점령 1개 | +2마리 | 레벨×10 |
| 병합 1회 | +4마리 | 레벨×10 |

## 4. 조작 방법 (Controls)

### 💻 PC
*   **디펜스 모드**:
    *   마우스 클릭: 탄환 발사 (적 없으면 현재 포탑 방향)
    *   스페이스바: 탄환 발사 (연사 가능)
    *   UI 클릭: SHIELD 토글, 선택지 선택
*   **마이닝 모드 (테트리스)**:
    *   `←` / `→`: 블록 좌우 이동
    *   `↑`: 블록 회전
    *   `↓`: 소프트 드롭
    *   `Space`: 하드 드롭
    *   `Enter`: 터미널 입력

### 📱 Mobile (Touch)
*   **디펜스 모드**:
    *   화면 터치: 탄환 발사 (멀티터치 지원)
    *   SHIELD 버튼: ON/OFF 토글
    *   입력창 직접 터치: 키보드 열림
*   **마이닝 모드**:
    *   가상 버튼: 좌/우/드롭

## 6. 기술 아키텍처 (Technical Architecture)

### 📂 파일 구조
```
js/modules/
├── GameManager.js      # 게임 전체 관리, 모드 전환, 저장/로드
├── DefenseGame.js      # 디펜스 모드 (2D Canvas), 적/아군/포탑/실드
├── TetrisGame.js       # 테트리스 모드 (Three.js 3D 원통)
├── TerminalUI.js       # 터미널 UI, 타이핑 효과, 명령어 처리
├── StageManager.js     # 맵/스테이지 관리, 접근성/점령 상태
├── InventoryManager.js # 인벤토리 관리, 아이템 드롭/획득
├── EquipmentManager.js # 장비 관리, 슬롯별 장착
├── ConquestManager.js  # 점령 관리, 스테이지 병합, 아군 강화
├── PerkManager.js      # 퍽 관리, 영구 강화
├── BGMManager.js       # 배경음악 관리, 볼륨 조절
└── ItemDatabase.js     # 아이템 데이터베이스

js/data/
└── virusDialogues.json # 바이러스 대사 데이터
```

### 🧱 레이어 구조 (Z-Index)

| Layer | Z-Index | 설명 |
|-------|---------|------|
| `#game-container` | 0 | 테트리스 (Three.js) |
| `defense-canvas` | 50 | 디펜스 (2D Canvas) |
| `#defense-ui` | 90 | 디펜스 UI (쉴드 버튼) |
| `#terminal-layer` | 100 | 터미널 (배경 투명, pointer-events: none) |
| `crt-overlay` | 9999 | CRT 스캔라인 효과 |

### 🛡️ 모바일 안정성
- **deltaTime 제한**: 최대 100ms (탭 전환 후 폭발 방지)
- **visibilitychange 감지**: 탭 복귀 시 상태 자동 복구
- **validateGameState**: 매 프레임 상태 유효성 검증
  - 코어 위치/HP 검증 및 리셋
  - 실드 HP 검증
  - 화면 밖 적/발사체 자동 제거
  - 화면 밖 아군 바이러스 재배치

## 7. 실드 시각 효과

### 상태별 애니메이션 (부드러운 보간)

| 상태 | 테두리 | 채우기 | 회전 |
|------|--------|--------|------|
| ACTIVE | 실선, 밝음 | 반투명 | 없음 (펄스) |
| OFF | 점선, 연함 | 없음 | 없음 |
| DISCHARGING | 점선 전환 중 | 사라짐 | 느린 회전 |
| CHARGING | 점선→실선 | 나타남 | 가속 회전 |
| BROKEN | 점선, 회색 | 없음 | 철컥철컥 |

### HP 기반 색상 그라데이션
- 100% HP: `rgb(0, 200, 255)` (파란색)
- 50% HP: 중간 보라색
- 1% HP: `rgb(255, 50, 50)` (빨간색)

## 8. 파밍 존 (Farming Zone)

### 특징
- **점령 불가**: 경유지 역할만 수행
- **무한 파밍**: 페이지 제한 없이 자원 획득 가능
- **타이머 표시**: 현재 페이지에서 생존 시간 실시간 표시
- **경유지 해금**: 방문 시 연결된 상위 스테이지 (Gamma, Delta) 접근 가능

### 진입 조건
- Safe Zone에서 Alpha 또는 Beta 점령 후 접근 가능
- 또는 Alpha/Beta를 통해 직접 이동
